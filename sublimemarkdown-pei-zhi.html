sublime+markdown 配置

周三 18 五月 2016
  1. 安装package control :https://packagecontrol.io/installation
  2. Ctrl+Shift+P 选择Package control :install package, 输入MarkdownEditing，Markdown Preview。
  3. 中文字体支持：安装package: ConvertToUTF8和GBK Encoding Support.
  4. 安装风格：Monokai Extended 设置：Preferences -> Color Scheme -> User -> Monokai Extended
  5. Markdown preview设置：
/* Sets the parser used for building markdown to HTML. */ 
"parser": "markdown", 
/* Enable or not mathjax support. */ 
"enable_mathjax": true, 
"enable_highlight": true, 
/* Remove github */ 
"enabled_parsers": ["markdown"]
}
  1. 浏览器预览（没成功）：Key-Bindings设置： {"keys": ["alt+m"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"}}
